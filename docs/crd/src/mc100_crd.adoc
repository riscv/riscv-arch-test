== Introduction

The MC100-32 processor certificate is based on the RVI20U32 profile with the addition of M-mode.
The MC100-64 processor certificate is based on the RVI20U64 profile with the addition of M-mode.

== Extensions

include::exts_intro.adoc[]

Unpriv extensions are aligned with the RVM Profile TG's latest thinking for its low-end microcontroller profile.
Priv ISA isn't availabe from the RVM Profile TG yet, but the extensions listed here are commonly supported by
microcontrollers and are supported by the certification tests.
