==== Zicclsm Misaligned Loads/Stores

Misaligned loads/stores can be handled in hardware if Zicclsm is implemented.  These tests exercise loads/stores in the program memory, which is expected to be cachable and coherent to support Zicclsm.  No special PMA circumstances are tested.  If other extensions with loads and stores are supported, more misaligned tests are run (e.g. ZicclsmF for floating point loads/stores, ZicclsmZca for compressed loads/stores).

==== ZihintntlZca Compressed Non-temporal Locality Hint

Compressed Zihintntl instructions `c.ntl.p1`, `c.ntl.pall`, `c.ntl.s1`, and `c.ntl.all` are a subset of the `c.add` hints. They are tested by the Zca_c_hint_cg/cp_chint_add coverpoint, so require no additional coverpoints.
