
=== Zicntr

These privileged tests exercise access to counters and HPMs (`time`, `cycle`, `instret`, `hpmcounter[31:3]`)
from lower privilege modes, as controlled by `mcounteren`, etc..
Note that the Zicntr and Ziphm unprivileged tests exercise access to these counters from M-mode
and the Sm tests exercise access to `mcycle`, `minstret`, `mhpmcounter`, etc. from M-mode.
