
=== Zicntr

These privileged tests exercise counters and HPMs reads (`time`, `cycle`, `instret`, `hpmcounter[31:3]`)
from lower privilege modes, as controlled by `mcounteren`, etc..
Note that the <<t-Zicntr-coverpoints, Zicntr>> and <<t-Zihpm-coverpoints, Zihpm>> unprivileged tests exercise access to these counters from M-mode
and the <<t-Sm-coverpoints, Sm>> tests exercise access to `mcycle`, `minstret`, `mhpmcounter`, etc. from M-mode.
